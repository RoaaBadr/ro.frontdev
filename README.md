# ro.frontdev

Personal portfolio website built with React, Vite, and Tailwind CSS.

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- React Router DOM
- React Icons

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Projects

The portfolio showcases Web Apps and UI/UX Design projects:

- **ShipConnect** — Logistics SaaS platform
- **Glamorix** — Beauty e-commerce store
- **Design Trends** — Personalized shopping UI/UX case study
- **OSMAUS** — Project management design concept
- **Brews & Co.** — Coffee shop branding
